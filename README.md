# VM Lifecycle Project

Общий репозиторий команды для работы с датасетами проекта по моделированию жизненного цикла и нагрузки виртуальных машин.

## Что лежит в репозитории

- `scripts/` - скрипты выгрузки и первичной обработки данных
- `notebooks/` или `.ipynb` файлы - исследовательские ноутбуки
- `docs/` или текстовые файлы - описание датасетов и переменных
- `README.md` - краткая инструкция по структуре и запуску

Большие сырые данные в репозиторий не загружаются.

## Что лежит вне GitHub

Сырые данные и крупные выгрузки хранятся отдельно:
- на внешнем накопителе
- в Google Drive
- в других согласованных папках команды

## Как запускать скрипты

Перед запуском нужно проверить пути к данным внутри `.py` файлов.

### macOS / Linux
Установка библиотек:
```bash
python3 -m pip install -r requirements.txt
```

### Windows
Установка библиотек:
```bash
py -m pip install -r requirements.txt
```

Запуск скрипта:
```bash
py scripts/ИМЯ_СКРИПТА.py
```

Если команда py не работает, можно использовать:
```bash
python scripts/ИМЯ_СКРИПТА.py
```
